## 函数作为参数

```python
def sum_sqaure(x):
    s = 0
    for _ in x:
        s += _**2
    return s

def sum_cube(x):
    s = 0
    for _ in x:
        s += _**3
    return s
```

```python
def sum(f, x):
    s = 0
    for _ in x:
        s += f(_)
    return s
```

```python
def sum(f, x):
    from operator import add
    return reduce(add, map(f, x))
```

## 函数作为返回值

```python
def compose(f, g):
    def h(x):
        return g(f(x))
    return h
```

```python
def curry(f):
    def g(x):
        def h(y):
            return f(x, y)
        return h
    return g

def uncurry(g):
    def f(x, y):
        return g(x)(y)
    return f
```
